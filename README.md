# Redis Cache Demo with Spring Boot

This is a **demo Spring Boot application** that demonstrates the usage of **Redis caching** in a simple CRUD and API setup.  

---

## Features

- Spring Boot application with REST APIs
- Redis caching for improved performance
- CRUD operations for Users, Employees, and other entities
- Separate services and repositories for clean architecture
- Demonstrates integration with Redis for caching data
- Example controllers for API testing

---

## Technologies Used

- Java 
- Spring Boot
- Redis
- Maven
- Docker (optional for running Redis)
- Git & GitHub

---

## Prerequisites

- Java 
- Maven
- Redis server (local or Docker)
- Git
